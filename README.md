## 📋 Descripción

El Oráculo de Bolsillo soluciona el problema de la "parálisis por análisis". Ya sea decidir qué comer, qué película ver o qué lenguaje de programación estudiar, esta app lo elige por ti con una animación satisfactoria tipo "Slot Machine".

## ✨ Características Principales

- **Gestión de Listas (CRUD):** Crea, edita y elimina tus propias categorías personalizadas.
- **Persistencia de Datos:** Todo se guarda automáticamente en el navegador (LocalStorage).
- **Audio Sintetizado:** Efectos de sonido generados en tiempo real con Web Audio API (sin archivos mp3 pesados).
- **Diseño Responsivo:** Funciona en PC y Móviles.
- **Animaciones CSS:** Efectos de desenfoque y rebote para una mejor experiencia de usuario.

## 🛠️ Tecnologías Usadas

- **HTML5:** Estructura semántica.
- **CSS3:** Grid Layout, Flexbox, Variables CSS y Animaciones Keyframe.
- **JavaScript (Vanilla):** Manipulación del DOM y lógica de estado.
